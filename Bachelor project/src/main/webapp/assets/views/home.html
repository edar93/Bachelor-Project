Scoring Application, part 2

This is skeleton for Scoring Database application. This project is part of CGI Java Boot Camp.

About template

This skeleton supports AngularJS (ver. 1.5.5) and JQuery (ver. 2.2.3).
Page layout is made by Bootstrap (ver. 3.3.6).

All dependencies are managed by Maven and Webjars. For details see main pom.xml file.

Embedded Tomcat server will start mvn tomcat7:run.
